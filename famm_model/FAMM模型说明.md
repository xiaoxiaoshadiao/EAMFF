# FAMM 模型代码框架设计说明书（基于论文4.2节）
## 一、模型核心目标
针对质子交换膜燃料电池（PEMFC）动态工况下的**短期波动退化预测**，捕捉电压序列中的瞬态扰动（如负载突变、水热波动）与背景状态演化，实现短步长（1-30步）高精度预测，为能源管理系统（EMS）提供实时功率分配依据。

核心依据：论文4.1-4.2节，模型输入为预处理后的电压序列 + P-M分数法筛选后的特征，输出为未来指定步长的电压预测值。

## 二、核心设计依据（严格对齐论文）
### 1. 模型整体架构
遵循 “多尺度分解 → 特征提取 → 门控融合” 三段式结构，对应论文公式4-1：
$$\hat{X}(n) = Gate(X_S(n), X_{bg}(n)) = f_{FAMM}(X(n))$$

| 符号          | 含义说明                                                                 |
|---------------|--------------------------------------------------------------------------|
| $X(n)$        | 输入序列（含电压及P-M筛选后的特征）                                      |
| $X_S(n)$      | 局部波动子序列（快速扰动，如负载突变、传感器噪声）                        |
| $X_{bg}(n)$   | 背景状态子序列（慢变趋势 + 历史恢复信息）                                |
| $\hat{X}(n)$  | 最终预测序列                                                             |

### 2. 关键模块设计约束
| 模块                | 论文核心要求                                                                 | 设计关键点                                                                 |
|---------------------|------------------------------------------------------------------------------|--------------------------------------------------------------------------|
| 多尺度分解模块      | 4.2.1节，双核平均池化+能量近似守恒，避免信号结构破坏                           | 输入序列拆分为局部项（$X_s = X - X_{bg}$）和背景项（$X_{bg}$=双池化均值） |
| 等距卷积模块（ICB） | 4.2.2节，编码→稳健分解→跨尺度建模→长度恢复，参数共享+跳跃连接                 | 双层ICB堆叠，含数值编码（VE）+位置编码（PE），等距卷积覆盖全局序列       |
| 门控融合机制        | 4.2.3节，GRU建模背景状态，逐时刻/逐通道自适应加权，权重满足单纯形约束         | $\alpha$ 由Sigmoid计算，融合公式：$\hat{X} = \alpha \odot Y_s + (1-\alpha) \odot Y_{bg}$ |

### 3. 输入输出规格
| 类型       | 具体要求                                                                 | 数据来源/格式                                                             |
|------------|--------------------------------------------------------------------------|--------------------------------------------------------------------------|
| 输入 $X(n)$ | 1. 序列长度 $I$：默认200-500个采样点（论文实验常用窗口）；<br>2. 特征维度 $F$：电压+P-M筛选后的关键特征（如电流、极板温度等，共6-8维）；<br>3. 数据预处理：需经过小波去噪（论文3.2.2节） | 虚拟生成数据（延续`generate_fuel_cell_data`逻辑，补充短期波动特征） |
| 输出 $\hat{X}(n)$ | 预测步长 $O$：1-30步（论文4.3节实验范围）；<br>输出维度：与输入特征维度一致（核心关注电压预测值） | 数组形状 $(O, F)$，重点评估电压预测的MAE/RMSE/$R^2$                        |

## 三、代码框架分层设计
### 1. 模块1：数据预处理与输入封装
#### 核心功能
- 加载/生成带短期波动的燃料电池虚拟数据（复用已有数据生成逻辑，新增高频扰动模拟负载突变）
- 滑动窗口划分序列（窗口大小=输入长度$I$），构建训练样本
- 7:3划分训练/测试集，对特征做Min-Max归一化（避免量纲干扰）

#### 关键约束
- 数据包含：慢变退化趋势 + 高频瞬态波动 + 低强度传感器噪声
- 窗口滑动步长：10个采样点（平衡数据量与计算效率）
- 归一化范围：[0,1]，按特征独立归一化（保留各特征自身分布特性）

### 2. 模块2：多尺度分解模块
#### 核心功能
实现输入序列的结构化拆分，满足能量近似守恒（论文公式4-6），分离“快速扰动”与“慢变趋势”。

#### 核心步骤（对应论文4.2.1节）
1. **双核平均池化计算背景项**：
   $$X_{bg} = mean(AvgPool_{k1}(Pad(X)), AvgPool_{k2}(Pad(X)))$$
   其中 $k1=3, k2=5$（工程常用核大小，论文未指定），Pad为零填充（保证池化后长度不变）。
2. **残差计算局部项**：
   $$X_s = X - X_{bg}$$（残差形式保留快速扰动，无信息丢失）
3. **能量守恒验证（可选）**：
   验证 $\|X\|_2^2 \approx \|X_{bg}\|_2^2 + \|X_s\|_2^2$，误差阈值<5%。

#### 输入输出
- 输入：$(batch, I, F)$（批量大小×序列长度×特征维度）
- 输出：$X_s (batch, I, F) + X_{bg} (batch, I, F)$

### 3. 模块3：等距卷积模块（ICB）
#### 核心功能
对局部项 $X_s$ 提取非线性波动特征，强化对瞬态扰动的敏感性（论文4.2.2节）。

#### 核心步骤（编码→分解→跨尺度→恢复）
1. **双重编码**：
   $$X_s^{eco} = Encoding(Concat(X_s, X_{zero}))$$
   - 数值编码（VE）：线性变换将特征映射到高维（编码维度$D$=64/128）
   - 位置编码（PE）：正弦位置编码，保留时间索引信息
   - $X_{zero}$：零填充（保证分解后长度一致）
2. **稳健分解**：复用多尺度分解逻辑，将 $X_s^{eco}$ 拆分为两个子序列
3. **跨尺度建模**：
   - 下采样卷积：$AvgPool_i$（下采样因子$i$=2）压缩长度
   - 等距卷积（IC1d）：卷积核覆盖全局序列，提取长距离依赖
   - 跳跃连接+层归一化+Dropout：抑制过拟合，保证梯度稳定
4. **长度恢复**：上采样（Conv1dTrans）+ 投影截断，输出与输入长度一致的局部特征 $Y_s$

#### 关键参数
| 参数名称         | 取值       | 说明                     |
|------------------|------------|--------------------------|
| 编码维度 $D$     | 64（默认） | 平衡精度与计算量         |
| 下采样因子 $i$   | 2          | 控制跨尺度建模粒度       |
| 激活函数         | Tanh       | 论文4.2.2节指定          |
| ICB堆叠层数      | 2          | 双层堆叠+参数共享        |

### 4. 模块4：背景状态建模模块（GRU）
#### 核心功能
对背景项 $X_{bg}$ 建模长期退化趋势与历史恢复信息（论文4.2.3节）。

#### 核心设计
- GRU层数：2层（参考实验最优配置）
- 隐藏层维度：与ICB编码维度$D$一致（保证融合时维度对齐）
- 正则化：隐藏层添加Dropout（rate=0.1），避免过拟合
- 输出：$Y_{bg}$（背景特征，形状与$Y_s$一致）

### 5. 模块5：门控融合模块
#### 核心功能
动态加权局部特征（$Y_s$）与背景特征（$Y_{bg}$），平衡扰动敏感性与趋势稳定性。

#### 核心步骤（对应论文4.2.3节）
1. **权重计算**：
   $$\alpha = \sigma(W_g [Y_s \| Y_{bg}] + b_g)$$
   - $[Y_s \| Y_{bg}]$：特征拼接（维度$2D$）
   - $W_g, b_g$：可学习参数，$\sigma$为Sigmoid（权重范围[0,1]）
2. **加权融合**：
   $$\hat{X}_{feat} = \alpha \odot Y_s + (1-\alpha) \odot Y_{bg}$$（逐时刻/逐通道计算）
3. **输出投影**：线性层将高维特征映射回原始维度$F$，得到最终预测序列 $\hat{X}$

### 6. 模块6：训练与预测流程
#### 训练相关（对齐论文4.3节实验设计）
| 配置项         | 取值/策略                                                                 |
|----------------|--------------------------------------------------------------------------|
| 优化器         | Adam（学习率=1e-4，权重衰减=1e-5）                                       |
| 损失函数       | MSE（对应论文预测误差$L_{pred}$）                                        |
| 早停机制       | 验证集RMSE连续5轮不下降则停止训练                                        |
| 批量大小       | 32（可适配GPU内存调整为16/64）                                           |
| 训练轮数       | 最大100轮（配合早停机制）                                                 |

#### 预测相关
- 单步预测：直接输出1步电压预测值
- 多步预测：滚动预测（每步预测结果作为下一步输入的一部分）
- 评估指标：MAE、MAPE、RMSE、$R^2$（论文4.3节指定）

### 7. 模块7：验证与可视化
#### 核心功能
验证模型性能，对齐论文实验结果展示要求。
1. **消融实验**：验证ICB模块、GRU模块、门控融合的必要性（对应论文表4.3）
2. **工况适应性验证**：在0A/1.78A/35.6A等不同电流工况下测试
3. **可视化输出**：
   - 预测值vs真实值曲线（整体序列+局部500点放大）
   - 各工况下的误差指标雷达图（与RAMM/EAMFF模型对比）
   - 门控权重$\alpha$的时间序列变化曲线（分析加权逻辑）

## 五、工程化注意事项
1. **数值稳定性**：多尺度分解前对输入序列归一化，避免数值溢出；池化/卷积后添加层归一化
2. **长度一致性**：所有模块输出需与输入长度一致，通过零填充（Pad）和截断（Truncate）保证
3. **梯度处理**：ICB模块使用跳跃连接，GRU使用梯度裁剪（clipnorm=1.0），避免梯度消失/爆炸
4. **兼容性**：输入特征维度可动态适配（支持6-8维特征，无需修改核心逻辑）
5. **可复现性**：设置全局随机种子（seed=42），保证实验结果可重复（论文4.3节要求）

1. **数据层**：复用`generate_fuel_cell_data`函数，新增“短期波动增强”（叠加高频噪声模拟负载突变）

## 八、端到端运行脚本（main.py）
### 8.1 脚本核心作用
作为FAMM模型的**一键式运行入口**，无需手动分步执行各模块，可快速串联“数据层→特征层→预处理层→模型训练→预测评估→可视化”全流程，既适配测试阶段的虚拟数据验证，也支持接入真实燃料电池数据（小波变换去噪后、PM得分筛选特征）的实际部署，降低模型落地使用门槛。

### 8.2 核心运行逻辑
脚本按“配置初始化→数据预处理（复用`generate_fuel_cell_data`/接入小波变换去噪结果）→特征适配（接收PM得分筛选的6-8维特征）→模型构建与训练（含早停/梯度裁剪）→测试集预测→多维度指标评估→结果可视化”的逻辑执行，全程自动化完成，无需人工干预中间环节。

### 8.3 关键可配置项
脚本内置核心可调参数，可根据实际场景灵活修改：
- 数据维度：调整生成/接入的数据样本量、输入序列窗口长度；
- 预测目标：修改预测步长（支持1-30步短期预测）；
- 训练策略：配置训练轮数、批次大小、学习率等；
- 可视化适配：指定需重点分析的特征（如Voltage），兼容不同维度特征的可视化需求。

### 8.4 运行方式
将脚本放置于`famm_model`目录下，确保依赖模块（`data_process.py`/`famm_core.py`等）均已就绪，通过命令行执行`python main.py`即可启动全流程，运行完成后自动输出评估指标、生成训练曲线/预测对比图/指标雷达图，且支持GPU/CPU自动适配。

### 8.5 适配性说明
脚本预留了与现有代码体系的衔接接口：
- 可直接替换`generate_fuel_cell_data`输出为真实燃料电池数据（小波变换去噪后）；
- 特征层可无缝接收PM得分筛选后的6-8维关键特征，无需修改核心逻辑；
- 预处理层兼容小波变换去噪后的序列输入，仅需调整数据接入路径即可适配。
