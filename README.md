EAMFF：面向动态工况的质子交换膜燃料电池多尺度退化预测与健康管理系统

研究背景与核心创新
1. 研究痛点
动态工况复杂性：频繁启停、负载波动、环境扰动导致 PEMFC 退化呈现多尺度特性（短期瞬时波动 + 长期非线性趋势）；
模型泛化难题：单一尺度模型难以适配不同退化模式，跨工况、跨数据集预测精度不足；
工程化鸿沟：学术模型与实际应用脱节，缺乏可落地的寿命评估与决策机制。

2. 核心创新点
（1）多尺度退化表征体系
从物理机理出发，构建覆盖 “短期扰动 - 背景状态 - 长期趋势” 的三维退化表征，首次将电压恢复行为纳入建模范畴，为多尺度预测提供理论基础。
（2）双专家模块协同设计
FAMM（波动感知多尺度模型）：通过多尺度分解 + 等距卷积 + 门控融合，精准捕捉短期瞬态退化特征，解决动态工况下的瞬时波动预测问题；
RAMM（恢复感知多尺度模型）：引入指数分解编码 + 历史信息整合，强化对长期非线性退化与电压恢复行为的建模能力。
（3）端到端自适应融合框架
提出 EAMFF 框架，通过轻量级路由策略实现不同退化模式的动态识别与模型切换，支持端到端推理与快速迁移微调，兼顾泛化能力与工程部署效率。
（4）工程化 RUL 评估体系
建立基于 “初始电压稳定区间均值 + 连续阈值判定” 的 RUL 评估流程，支持动态阈值配置（3.0%~4.5% 相对衰减），抑制噪声干扰，提升工程场景的稳健性。
