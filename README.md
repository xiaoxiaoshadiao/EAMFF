质子交换膜燃料电池（Proton Exchange Membrane Fuel Cells, PEMFCs）因其高效率、零排放和快速动态响应等优势，正在成为交通运输、分布式发电和储能系统中的重要绿色能源技术。然而，在复杂动态工况下，频繁启停、负载波动以及环境扰动会显著加速关键部件的退化，导致性能衰减与寿命缩短，严重制约了工程化应用与规模化推广。
针对这一问题，本文围绕PEMFC在动态工况下的退化特性，系统开展了多尺度退化建模与预测方法研究，旨在提升燃料电池的可靠性，并为能源管理系统（Energy Management System, EMS）的优化提供理论与技术支撑。首先，本文从物理机制角度分析了PEMFC的关键退化机理，并结合实验数据构建了覆盖短期扰动与长期趋势的多尺度退化表征体系。在此基础上，设计了两类预测模型：面向短期波动的预测模型(Fluctuation-Aware Multi-scale Model, FAMM)，通过多尺度分解、卷积与门控融合机制，有效刻画瞬态退化特征；考虑电压恢复的预测模型(Recovery-Aware Multi-scale Model, RAMM)，引入指数分解与历史信息整合模块，提升了对长期非线性趋势的建模能力。
进一步地，本文提出了端到端自适应多尺度融合预测方法(End-to-End Adaptive Multi-Scale Fusion Framework, EAMFF)。该方法通过轻量级结构实现不同退化模式的动态识别与模型切换，增强了系统的泛化能力与工程适应性。它既支持端到端推理，在未见样本条件下进行退化趋势预测，也可通过快速迁移微调适配不同平台与复杂工况，从而兼顾模型泛化与工程部署。实验结果表明，所提出的方法在预测精度、稳定性和可解释性方面均优于现有主流模型，能够有效支撑车载能源管理系统的实时决策与长期调度。
综上，本文工作不仅为PEMFC性能退化建模与健康状态预测提供了系统性解决方案，也在理论上丰富了多尺度退化预测方法体系，在工程上提升了燃料电池的可靠性，对实现绿色交通和能源转型具有重要意义。


本文的主要创新点可归纳为以下几个方面：
（1）提出了一套简洁的数据分析流程。针对PEMFC动态工况复杂、多源耦合的特点，设计了轻量可解释的P-M分数法进行特征筛选，无需复杂机理建模，能有效融合变量间的线性与非线性依赖信息，同时采用稳定的小波分析进行噪声抑制。该流程以清晰的步骤和较低的计算代价，实现了从原始数据到敏感特征的可靠提取，为后续建模提供了高质量、易处理的数据基础。
（2）建立了一种面向动态工况的多尺度退化建模假设。为克服复杂机理统一建模的困难，本文从工程可实现性出发，将输出电压序列显式分解为短期波动序列、长期趋势序列及噪声序列。该假设为多尺度“分而治之”的建模策略提供了清晰且物理可解释的理论框架。
（3）构建了一个兼顾短期波动与长期趋势的专用预测模型体系。基于上述假设，针对性设计了FAMM与RAMM两个模型：FAMM通过“分解——等距卷积——门控融合”结构精准捕捉快速动态响应；RAMM通过“指数分解——历史整合”机制有效刻画长期衰退与恢复现象。二者在各自目标尺度上实现了建模精度与效率的平衡。
（4）设计了一种统一的端到端自适应多尺度预测方法EAMFF。采用混合专家设计思路，将不同尺度预测能力模块化，依赖步长一致性路由机制动态调度专家，以及采用分量一致性损失，保障整体方法协同训练的稳定性。该方法在结构统一的前提下，实现了预测逻辑的自适应与跨工况的强泛化能力。
（5）完成了系统性的实验验证与工程化评估流程。通过系统的消融实验验证了文中各类算法模块的必要性，通过跨数据集与少样本迁移实验证明了EAMFF方法的泛化性，并构建了包含工程化失效阈值定义、在线滚动预测、RUL误差量化在内的完整寿命评估流程，全面验证了方法在工程应用角度的可行性。

尽管本文在PEMFC多尺度退化预测与工程化应用方面取得了一定进展，但仍认识到其在迈向更可靠的能源系统智能管理进程中存在深化空间。未来工作可围绕以下方向开展：
（1）建立不确定性感知的预测——决策框架。当前预测虽精度较高，但仍缺乏对预测结果置信度的量化描述。未来可引入贝叶斯神经网络、深度概率模型等方法，输出具有统计意义的预测区间，从而支撑能源管理系统在风险感知条件下的鲁棒优化与预防性调度，提升整体系统的决策韧性与安全性。
（2）探索机理嵌入的混合智能建模范式。为进一步增强模型在未知工况与极端状态下的外推能力及物理可解释性，可尝试将燃料电池的守恒方程、老化经验公式等以软约束或结构先验形式嵌入深度学习框架，发展物理信息机器学习在电化学系统建模中的应用路径，实现数据驱动灵活性与机理一致性的统一。
（3）构建系统级协同健康管理与寿命均衡策略。面向多电堆、多能源构成的实际车用或储能系统，可将本预测方法扩展至系统层级，研究电堆间退化耦合机制、寿命分布特性与协同能量管理策略，形成基于全系统健康状态的动态功率分配与维护规划方法，提升整体系统的经济性与可靠性。
（4）推动算法——硬件——系统的闭环验证与落地迭代。最终需在真实车载或储能平台开展长期在线部署与闭环验证，在实时数据流、边缘计算约束及实际工况波动下，持续评估模型的稳定性、自适应能力与寿命预测准确性，形成“监测——预测——控制——维护”一体化系统解决方案，完成从方法创新到工程价值的闭环。
