EAMFF：燃料电池电压预测与剩余寿命（RUL）评估系统
项目概述
本项目针对燃料电池的多步长电压预测与剩余寿命（RUL）工程化评估需求，提出了基于专家分量封装的 EAMFF（Expert-Aware Multi-Scale Fusion Framework）模型。该模型通过分解 - 融合策略捕捉电压退化的局部扰动、背景状态与长期趋势，在跨数据集、跨工况场景下实现高精度预测，并通过工程化的 RUL 评估流程，为实际应用提供可复现、高稳健性的寿命判定方案。
核心目标：
解决燃料电池电压序列的多步长（10~150 步）预测问题；
实现工程化的 RUL 识别与精度评估，支持动态失效阈值配置；
验证模型在不同退化速率、不同数据集下的泛化能力。
模型架构
EAMFF 模型采用「分解 - 封装 - 融合」的三段式架构，核心模块如下：
1. 专家分量封装模块（ExpertEncapsulation）
整合两个专项专家模块，分别捕捉不同尺度的退化特征：
FAMM（局部 - 背景分解模块）：通过多尺度分解（MultiScaleDecompose）+ 等距卷积块（ICB）+ GRU 背景建模，输出局部扰动分量（Xs）和背景状态分量（Xbg）；
RAMM（长期趋势建模模块）：通过多尺度指数分解编码（MEDEM）+ 历史信息整合（HIIM）+ 融合预测（MPM），输出长期恢复分量（Xr）；
维度统一层：确保三路分量维度一致，为后续融合提供基础。
2. 门控路由与融合预测
门控权重网络：根据预测步长动态调整三路分量的融合权重（α∈[0,1]），适配不同时间尺度的预测需求；
融合层：通过特征拼接与线性投影，输出最终预测结果，兼顾精度与计算效率。
3. 工程化 RUL 评估模块
失效阈值定义：基于初始电压（V_init）的相对衰减比例（3.0%~4.5%），适配不同数据集特性；
寿命终止判定：连续 5 个采样点低于阈值时触发失效判定，抑制噪声干扰；
滚动预测策略：单步预测构建 RUL 序列，模拟在线评估场景。
